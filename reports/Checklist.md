# 📋 Чек-лист тестирования приложения




## 🏠 Главная страница
- Отображение списка последних новостей
- Корректный переход на страницу «Все новости» по ссылке/кнопке
- Скрытие и раскрытие дополнительной информации о новости

## 📰 Раздел «Новости»
- Отображение полного списка всех новостей
- Работа пагинации или подгрузки
- Скрытие и раскрытие информации о новости
- Работа фильтров
- Работа сортировки
- Переход в «Панель управления»

## ⚙️ Панель управления
- Успешное добавление новой новости
- Удаление новости
- Редактирование существующей новости и сохранение изменений
- Работа фильтров и сортировки в списке новостей панели управления

## ✏️ Создание / Редактирование новости
- Валидация обязательных полей
- Валидация форматов данных
- Отображения ошибок валидации пользователю
- Успешное сохранение новости после ввода корректных данных
- Отмена создания/редактирования без сохранения

## 💬 Раздел «Цитаты»
- Отображение списка всех цитат
- Работа пагинации или подгрузки
- Скрытие и раскрытие дополнительной информации о цитате

## ℹ️ О приложении
- Отображение текущей версии приложения
- Работоспособность всех ссылок

## 🛡️ Общие проверки (Non-functional)
- Переход по всем страницам через главное меню навигации
- Поворот экрана: проверка адаптивности и отсутствия артефактов верстки
- Работа с сетью: отключение, включение, переключение типа сети
- Сохранение состояния приложения после прерывания
- Отображение сообщений об ошибках
- Обработка системной кнопки "Назад" и навигационных жестов
- Соответствие интерфейса утвержденным дизайн-макетам
- Удобство использования: читаемость текста, достаточный размер кликабельных областей
